KaththiSandai is a gamified web application designed to transform that struggle into a strategic Sword Fight, 
turning the "grind" of exam prep into an immersive, competitive experience.

In this arena, students aren't just learners—they are Warriors. 
By replacing traditional test-taking interfaces with a Battlefield theme, 
KaththiSandai leverages the psychological benefits of gamification to increase student engagement and retention.

Core Features
The Battlefield: A complete gaming-inspired UI that transforms boring assessments into high-stakes duels.

Warrior Profiles: Personal user accounts to track progress, stats, and historical performance.

Global Leaderboard: Real-time rankings to foster healthy competition among students nationwide.

Dynamic Question Generation: High-fidelity questions tailored specifically for Boards, NEET, and JEE.

The Tech Stack
This project was architected and deployed within a 7-day sprint, showcasing the power of rapid AI-assisted development:

AI Engine: Gemini 2.5 Flash for high-speed, intelligent question generation.

Orchestration: LangChain for managing complex AI agent workflows and prompt sequencing.

Data Integrity: Structured Prompting to ensure all AI outputs are delivered in precise JSON format for seamless backend integration.

Deployment: Hosted on PythonAnywhere for stable, accessible web performance.
